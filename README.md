
# Staff Management Floor Plan — Front-End Project

##  Description
Application web interactive permettant de gérer le placement des employés
dans un plan d’étage, avec règles d’accès, profils détaillés et interface intuitive.


##  Features principales
- Ajout d’employés via modale
- Formulaire dynamique pour les expériences
- Zones avec restrictions par rôle
- Affectation et désaffectation d’employés
- Profil détaillé cliquable
- Responsive complet (Desktop, Tablet, Mobile)
- Sauvegarde dans localStorage
- Animations fluides
- Drag & Drop (si implémenté)

##  UI / UX
- Palette cohérente
- Icônes intuitives
- Zones visuelles claires
- Mobile-first

## Expérience Utilisateur

- Design responsive (Mobile, Tablet, Desktop)

- Animations fluides et transitions

- Interface intuitive avec icônes explicites

- Palette de couleurs cohérente et professionnelle

##  Technologies utilisées
- HTML5
- CSS3 (Flexbox, Grid, Animations, Media Queries)
- JavaScript (DOM, Events, LocalStorage)
